# 📱 Login App

## 🚀 Overview

The Login App is a simple Android application that allows users to log in using email and password. It includes input validation and basic authentication logic.

---

## ✨ Features

* 🔐 User login with Email & Password
* ✅ Input validation
* ⚠️ Error handling for invalid credentials
* 🎨 Clean UI using Material Design

---

## 🛠 Tech Stack

* Java
* XML
* Android Studio

## ⚙️ How It Works

1. User enters email and password
2. Input is validated
3. Credentials are checked
4. If correct → navigate to Home screen
5. If incorrect → show error message

---

## ⚙️ How to Run

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Run on Emulator or Device
